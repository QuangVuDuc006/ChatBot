FAQ

Answers before you sign in.

What happens when I try the workspace?

You land on a short login screen, then open the chat workspace where you can start with a question, draft, text file, or image.

Can I keep useful conversations?

Yes. Conversations are saved locally in the browser so you can return to useful threads, rename them, and keep related work together.

Can I use files or images as context?

Yes. Attach text files or supported images when the answer depends on source material, screenshots, notes, or drafts.

Do I need to set up a full account system?

No. This build uses a lightweight entry screen and local chat history, so you can evaluate the workspace without adding a user database first.

Are developer/API routes still available?

Yes. The chat page remains at /chat, and the message API still accepts requests at POST /chat.
